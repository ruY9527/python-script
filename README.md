| 模块名字   | 模块作用                | 备注说明 |
| ---------- | ----------------------- | -------- |
| basic      | 基础模块和语法的学习    |          |
| datafiles  | 相关数据库文件阅读      |          |
| db         | 操作数据库相关          |          |
| files      | 操作文件相关            |          |
| flask      | flask服务框架运行       |          |
| geo        | 地理空间操作            |          |
| images     | 操作图片相关            |          |
| middleware | 操作kafka,redis等中间件 |          |
| pandas     | pandas操作相关          |          |
| pdfwords   | 操作pdfwords相关文件    |          |
